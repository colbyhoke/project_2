Welcome to the COVID API documentation

Below, you'll find links and examples for how to use our API. The results are returned as JSONified data.

Data last updated: 10/5/2020

Cases and deaths

COVID-19 cases and deaths county data gathered from the New York Times COVID-19 project on GitHub.



Data structure example

"cases": 1,
"county": "Wake",
"date": "2020-03-03",
"deaths": 0,
"fips": 37183,
"state": "North Carolina"

Filter by: state

Append the two-letter state abbreviation after the slash.

For example, for North Carolina, use: covid-all/nc
